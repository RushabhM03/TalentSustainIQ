Aspects

Heres a look at the training and prediction APIs

/training

TRAINING

Api for training the model by selecting the best model based on accuracy score.

Includes all the steps like preprocessing, finding best models, data cleaning

Run on POSTMAN: http://0.0.0.0:5000/training with the training data in the correct folder

/batchprediction

BATCHPREDICT

API used to make prediction for an entire batch of data

Typically, this API is used for testing purposes

Run on POSTMAN: http://0.0.0.0:5000/batchprediction with the data in the correct folder

/prediction

SINGLEPREDICT

This API is used by the form to make predictions on the trained model using data from the form

More frequently used than batchpredict API

Run on POSTMAN: http://0.0.0.0:5000/prediction with the data in the request header

Project Setup

STEP 1

Clone the project and setup the virtual environment and install dependency packages

STEP 2

Put the data in the training_data and prediction_data folders. Note that the data should be without any errors

STEP 3

Run the training API on POSTMAN as well as the batchprediction API

STEP 4

Open the localhost port 5000 and run the User interface form for predicting the likelyness of an employee at staying in the company

FINISH

Project is setup. Now you can retrain it as per your own convinience

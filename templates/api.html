API Management


View APIs here and use them to train/test the model using your own custom data

API Description
/predict_client UI form for making single predictions which takes the required fields as input
/training API for training the model using your own custom dataset and selecting the best ml model
/batchprediction API to test the trained model on a batch of data which is to be placed in the prediction_data folder
/prediction API to make single predictions on a specific row of data
